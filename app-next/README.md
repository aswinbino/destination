# Destination Explorer (Next.js)

This is the Next.js-based component of the Destination application, designed for high-performance browsing and search.

## Features
- Server-side rendering (SSR) for faster initial loads.
- Advanced state management with Zustand.
- Data fetching with TanStack Query.

## Setup
```bash
npm install
npm run dev
```

## Deployment
Deploy as a standard Next.js app to Vercel.
